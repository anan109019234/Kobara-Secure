WHAT IS KOBARA SECURE?

Kobara Secure (Web Version) is a simple web app I built to help people check if a link is safe or not. It uses a machine learning model (Random Forest) trained on phishing data, and the app itself runs on Streamlit. You just paste the link, click detect, and it tells you right away whether it’s phishing or safe. There's also a history section so you can look back at previous checks. Then I took it a step further and made Kobara as a Chrome extension. The idea came from realizing how easy it is to click on shady links without thinking. So I built this tool that lets you check URLs straight from your browser—either by typing them into a popup or just right-clicking on any link. It connects to a Flask server that runs the same machine learning model behind the scenes. You get instant results, a confidence score, and a short explanation. The project really helped me learn more about backend stuff and Chrome extension APIs—and honestly, it’s just cool to see an idea like this actually working. If you're into browser-based AI tools too, I’d be happy to chat or collaborate!

Installation Guide

  1. Download the kobara_secure_extension.rar file.
  2. Extract the contents of the RAR file to a folder on your computer.
  3. Open your browser's extensions management page (e.g., `chrome://extensions` for Chrome).
  4. Enable "Developer mode" (usually a toggle in the top right).
  5. Click "Load unpacked" and select the extracted extension folder.
  6. The Kobara Secure extension should now be active!
