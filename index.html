This project came from a simple idea: I often come across suspicious links shared in group chats or emails, and I thought, "Wouldn't it be great if there was a simple app that could help people check whether a link is safe before they click on it?". That’s how I started working on this project – a web-based phishing URL detection app using Streamlit and a Random Forest machine learning model trained on phishing datasets.

Installation Guide

  1. Download the kobara_secure_extension.rar file.
  2. Extract the contents of the RAR file to a folder on your computer.
  3. Open your browser's extensions management page (e.g., `chrome://extensions` for Chrome).
  4. Enable "Developer mode" (usually a toggle in the top right).
  5. Click "Load unpacked" and select the extracted extension folder.
  6. The Kobara Secure extension should now be active!
